# Malcolm Knowles — 성인 학습 이론 (Andragogy)

**Malcolm Knowles** | *The Modern Practice of Adult Education: Andragogy versus Pedagogy* (1970)

이 자료는 하네스 엔지니어링과 직접적인 관련은 없다. **이 워크숍의 교육 설계 방식**의 근거다.

---

## 핵심 원칙

성인은 **자신의 필요를 인식할 때** 학습 동기가 생긴다.

원리를 먼저 배우면 "그래서 뭐에 쓰지?"가 남는다. 문제를 먼저 가지고 있으면 원리를 배울 때 "아, 이걸로 내 문제를 풀 수 있겠다"가 된다.

---

## 워크숍 설계에서의 적용

| Knowles 원칙 | 이 워크숍에서의 구현 |
|---|---|
| 문제 중심 학습 | 블록 1: 불편함 먼저 뽑기 |
| 경험 기반 학습 | 블록 2: 원리를 설명하기 전에 공통 코드베이스로 직접 체험 |
| 자기주도 학습 | 블록 3: 과제 카드 스스로 선택, 자율 적용 |
| 즉각적 적용 가능성 | 블록 4: "내일 당장 할 수 있는 것" 1가지 확정 |

블록 1이 맨 앞에 있는 이유, 블록 2에서 강의보다 실습 비중이 높은 이유가 여기 있다.
