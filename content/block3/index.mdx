## ✅ 모든 카드의 공통 완료 조건

카드와 무관하게 아래를 목표로 합니다:

- [ ] 오늘 작성한 `CLAUDE.md` / `AGENTS.md` / Rules 에 "자주 하는 실수" 항목이 1개 이상 추가됐다
- [ ] 오늘 시도한 것을 회고 시간에 3분 안에 설명할 수 있다
- [ ] "내일 이 방법으로 실제 업무에 써볼 수 있다"고 말할 수 있다
